# Project Spec

## Tech Stack

-   **API Client**: [Axios](https://axios-http.com/)
-   **Styling**: [TailwindCSS](https://tailwindcss.com/)

## Project Structure

```
/src
 ├─ main.jsx          # Entry point
 ├─ libs
 │   └─ http_client   # API client setup (axios instance)
 └─ features          # App features/modules
```

## Development

```bash
# install deps
npm install

# run dev server
npm run dev

# build for production
npm run build
```
